## ActiveWorlds Extension commands

These commands were added by the ActiveWorlds software, and are not normally supported by generic Renderware software. If they are used in RWX files that should be usable by other software then ActiveWorlds, they should be prefixed by a crunchbang ("#!") so they are ignored by software not supporting them.
